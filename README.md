# Projetos de Sistemas Eletrônicos — UNESC

Coleção de projetos desenvolvidos na disciplina **Sistemas Eletrônicos** da UNESC, utilizando o software de simulação [Proteus](https://www.labcenter.com/).

---

## Sobre

Os projetos cobrem desde circuitos básicos, como controle de motor e LED, até implementações mais complexas, como uma fonte linear de tensão regulada. Para visualizar os arquivos, é necessário ter o Proteus instalado.
